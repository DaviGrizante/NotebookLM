# Projeto DIO de NotebookLM

O assunto escolhido para o projeto foi Banco de Dados, meu objetivo era utilizar o Notebook como uma fonte de estudos para os conceitos fundamentais e principais de Banco de dados e poder receber sugestões e auxílio para a realização de projetos práticos.

# Principais Fontes

A IA consumiu diversas fontes diferentes, entre elas vídeos no youtube e pdfs que inseri manualmente, o restante das fontes foram inseridas através do sistema de deep research, sendo elas sites do google. Os pdfs que inseri foram:
- [Introdução a Banco de Dados](https://www.ime.usp.br/~jef/apostila.pdf)
- [Introdução a Sistemas de Bancos de Dados](https://www.kufunda.net/publicdocs/Introdu%C3%A7%C3%A3o%20a%20Sistemas%20de%20Bancos%20de%20Dados%20(C.%20J.%20Date)%20(z-lib.org).pdf)
- [Banco de Dados - Projeto e Implementação](https://github.com/VinnyLima/livros/blob/master/Banco%20de%20Dados%20-%20Projeto%20e%20Implementa%C3%A7%C3%A3o.pdf)

# Prompts

## Prompt Inicial
O prompt que usei foi: "Você é um professor de TI da faculdade e precisa passar uma aula introdutória aos alunos sobre Banco de dados, explicando os conceitos fundamentais e utilizando exemplos para facilitar o entendimento deles. Redija um texto base para estruturar essa aula". A partir daí, a IA me explicou todos os principais conceitos de banco de dados, partindo do princípio diferenciando dado de informação e finalizando com uma introdução ao SQL, com um exemplo prático ao fim sobre um sistema de biblioteca. 

## SQL
Após isso, meus prompts se basearam nesse primeiro, portanto, continuei na ideia da IA ser um professor e ter que dar uma aula sobre banco de dados, porém com outros conceitos. Pedi para ele dar exemplos práticos de operações CRUD com SQL em um SGBD. A própria IA escolheu o MySQL como SGBD por entender que é uma das mais populares do mercado e resolveu simular um gerenciamento de supermercado. Então a IA começou a explicar sobre DLL, DML, cada uma das operações do CRUD e tudo isso com exemplos práticos no próprio SQL. Ao final, ele ainda sugeriu um próximo tema para estudar, o JOIN.

## Projeto
Para finalizar, pedi para IA implementar um projeto com uma linguagem de sua escolha e integrasse o banco de dados, a IA escolheu Java e explicou passo a passo todos os conceitos de uma conexão de banco de dados MySQL com java.

## Sobre os Prompts
Todos os prompts foram utilizados de forma genérica, mas é sempre possível pedir para a IA utilizar ferramentas diferentes e exemplos ou sistemas específicos para ter mais controle sobre a resposta, por exemplo, solicitar um SGBD diferente de MySQL ou uma linguagem diferente de Java.
